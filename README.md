# Bank Marketing Excel Dashboard

## Project Overview
This project presents a data analysis dashboard built using bank marketing campaign data in Excel. It focuses on identifying customer behavior patterns and key factors influencing term deposit subscriptions.

## Objective
- Analyze customer response trends  
- Identify high-performing customer segments  
- Support better marketing decision-making  

## Dataset
- Bank Marketing Dataset (Kaggle)  
- Contains customer demographics, campaign details, and response data  

## Key Insights
- Job Role: Management and blue-collar show higher subscriptions  
- Age Group: 25–45 customers are most responsive  
- Education: Majority have secondary education  
- Marital Status: Married customers subscribe more  
- Contact Type: Cellular performs better than telephone  
- Campaign Calls: Moderate calls give better results  
- Balance: Higher balance customers are more likely to subscribe  

## Features
- Subscription analysis by job, age, and month  
- Customer segmentation based on education and marital status  
- Campaign performance using contact type  
- Loan impact analysis (housing & personal loan)  
- Comparison of job vs average balance  

## Business Value
- Helps identify target customers  
- Improves campaign efficiency  
- Supports data-driven decision making
